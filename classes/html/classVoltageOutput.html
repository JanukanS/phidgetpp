Detailed Description

Class for the Isolated Voltage Output phidget.

Class interfaces with the C API for the Isolated Voltage Output (OUT1001_0). It is recommended to define all devices in a namespace above the main function. Still need to implement software voltage limits

Constructor sets the hub port and serial number if defined and attempts to connect to the device. Serial Numbers are needed for identification when connecting multiple identical devices.
