\subsection{Detailed Description}
Class for the Graphic L\+CD Phidget. 

Class interfaces with the C A\+PI for the Graphic L\+CD Phidget (L\+C\+D1100\+\_\+0). It is recommended to define all devices in a namespace above the main function.
