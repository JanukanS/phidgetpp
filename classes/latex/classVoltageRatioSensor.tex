Class for the \hyperlink{classVoltageRatioSensor}{Voltage\+Ratio\+Sensor} Phidget. 

Class interfaces with the C A\+PI for the Voltage Ratio Sensor on the hub (H\+U\+B1000\+\_\+0). It is recommended to define all devices in a namespace above the main function.
